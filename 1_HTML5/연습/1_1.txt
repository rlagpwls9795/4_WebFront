<!DOCTYPE  html> : 문서 형식을 지정하는 태그
<html> : html 문서 내용을 정의하는 태그, 이 안에 html 코드 작성
         lang 속성 - html 문서가 어떤 언어로 되어있는지 표시, 검색 엔진이 참고하는 용도
<head> : html 문서를 정의하는 태그 (문서의 제목, 스타일시트, 스크립트, 메타데이터)
<meta charset="UTF-8"> : html 문서의 문자 형식(문자 인코딩) 선언
<body> : 화면에 보여질 html 문서 내용을 작성하는 태그
<h1~h6> : 제목을 입력할 때 사용하는 태그
<hr> : 수평선 태그

html은 연속되는 띄어쓰기는 1칸 띄어쓰기로 인식, 엔터는 인식되지 않는다.
띄어쓰기, 엔터를 html에 표기하는 방법
1. 띄어쓰기 : &nbsp;
2. 엔터 : <br>

<p> : 글 작성용 태그, 문단 구분(앞, 뒤 태그와의 간격 생성)

<pre> : 글 작성용 태그, 문단 구분(앞, 뒤 태그와의 간격 생성) 
         + 시작, 종료 태그 사이에 작성된 문자열의 형식을 그대로 화면에 표현(출력)

=============================================================================================
<b> : 단순히 글자를 굵게 하는 태그
<i> : 단순히 글자를 기울이는 태그 

<strong> : 중요성, 심각함 등을 표현하는 강조 태그
<em> : 문장 일부분의 중요성을 강조할 때 사용하는 태그 (기울어짐)

 b, i와 strong, em의 차이
    1) 태그 자체의 의미의 차이
    2) 웹 접근성의 차이
* 웹 접근성 : 일반인 뿐만 아니라 장애인, 고령자를 위해
              웹 사이트에서 제공하는 서비스를 모두 이용할 수 있도록 보장하는 것(의무 사항)  

<mark> : 형광펜 효과
<u> : under line 
<del> : 삭제선 (없어진 텍스트)
<small> : 작은 글자 표기
<sup> : 윗첨자
<sub> : 아랫첨자
<abbr> : 마우스 오버 시 툴팁 출력

=============================================================================================
<ul> : 순서가 없는 목록 태그
<ol> : 순서가 있는 목록 태그 (type-a,A,i,I,1)
<li> : ul, ol 태그 내부에 들어 갈 목록의 요소를 나타내는 태그

table 태그 
- 웹 문서에서 자료 정리를 위해 주로 사용하는 태그 
- 행과 열로 이루어져 있으며 행과 열이 교차하는 지점을 cell이라고 함
- 테이블을 나타내는 행과 열이 작성될 수 있는 영역을 지정함

<tr> : table row, 한 행
<td> : table data, 한 행의 한 컬럼(셀)
<th> : table header, 컬럼명 표시, 기본 성질은 <td>와 같으나 굵은 글씨, 가운데 정렬
<caption> : 테이블의 제목이나 설명 내용 추가하는 태그

td/tr 태그에 작성 가능한 속성
- 행 병합(rowspan) : 상하 병합
- 열 병합(colspan) : 좌우 병합

테이블 구조 설정 태그 : thead, tbody, tfoot

영역 관련 태그
1) block 
- 공간을 수직 분할 (수직으로 되어 있는 화면에 가로선을 그어 여러 행으로 분할)
- 크기를 지정하는 css 속성인 width, height를 사용할 수 있다 
ex) p/pre, h1~h6, hr, div

2) inline
- 공간을 수평 분할 (수평으로 되어 있는 화면에 세로선을 그어 여러 열로 분할)
- 크기를 지정하는 css 속성인 width, height를 사용할 수 없다
ex) strong/b, em/i, mark, span

block과 inline의 차이점
1) 줄 바꿈(영역 분할 방식) 
    : block 형식 태그는 한 줄을 모두 차지하기 때문에 뒤에 작성되는 내용은 다음 줄에 출력
      inline 형식 태그는 한 줄 내에서 작성된 내용 만큼의 칸을 차지하기 때문에 다음에 작성된 내용이 뒤에 붙어 출력
      *inline 태그 작성 시 엔터를 입력하면 출력하는 화면에서 한 칸 띄어쓰기 된 형태로 출력

2) 영역 지정 방식
    : block 태그는 사각형 박스 형태로 지정
      inline 태그는 내부에 작성된 내용 단위로 지정

3) 크기 지정
    : block 태그는 크기 지정 가능
      inline 태그는 크기 지정 불가능

iframe 태그 : 웹 문서 내부에 다른 웹 문서를 추가하는 태그 (inline 형식) 
              속성 - src="url"

<img> 이미지  태그
- 웹 문서에 사진이나 그림 같은 이미지 삽입하는 용도의 태그
- 내용이 없기 때문에 종료 태그 없음
- inline 형식
[속성]
1) src : 삽입 이미지의 경로 지정
2) width/height : 이미지의 크기
3) alt : 이미지 설명 (이미지가 출력되지 않는 경우, 웹 접근성)

하이퍼링크 태그 

하이퍼링크 : 다른 문서로 즉시 접근하는 텍스트
HTTP : 하이퍼텍스트로 이루어진 문서를 전송하는 규약 (HTML)
<a> : 현재 HTML 문서와 다른 HTML 문서를 연결해서 이동할 수 있게 하는 기능을 제공하는 태그(페이지 이동)
      <a>로 감싸진 문자열 또는 이미지 ==하이퍼링크
      inline 형식 
** 하이퍼링크는 웹 문서가 다른 문서와 구분될 수 있는 핵심 기능
  1) 다른 페이지로 이동
  2) 현재 페이지 내에서 이동 

[속성]
1) href : 참조할 페이지 주소(이동할 페이지 주소)를 작성하는 속성
2) target : 연결된 문서를 어디에 열 것인지 지정하는 속성 
            _self : 현재 탭(기본값)
            _blank : 새 탭(새 창)
